Analizadores fijos o permanentes

Dranetz es prionero en sistemas de monitoreo de calidad de energía permanente para isntalar en rack o en celdas de media o baja tensión. El sistema se complementa con software Encore, basado en el navegador web, incluye la familia 61000 (61STD, 61SG, 61SGD) y es el primer instrumento completamente modular y configurable que rompe el formato de instrumento tradicional de 8 canales (4 tensiones / 4 corrientes).

Usted puede elegir entre los módulos de adquisición de tensión, corriente y datos para contruir de uno a cuatro instrumentos en un solo formato compacto y rentable

Software

Características

El software DRAN-VIEW7 es por mucho la mejor herramienta software de análisis que permite a los profesionales de la energía visualizar y análizar de forma simple y rápida los datos de monitoreo de energía. Es fácil de navegar, brinda funcionalidad automatizada e incorpora potentes capacidades de analíticas y opciones personalizables para satisfacer las necesidades de cada usuario individual.

Dranview permite conectarse a los equipos Dranetz HDPQ y controlarlos de manera remota vía VNC, incluyendo tambinén la funcionalidad de descarga remota de infromación.

Características

Encore es un software avanzado, basado en la web que es un administrador de los equipos y una interfaz de usuario para el análisis de los parámetros y eventos de calidad de energía. El software Encore se puede utilizar en cualquier aplicación, desde pequeños con pocos equipos hasta enormes sistemas de monitoreo multipropósito para empresas gigantes o empresas de servicios públicos con 50 o más puntos. Encore Software se comunica automáticamente con cada equipo de monitoreo a través de todos los métodos de comunicación compatibles para descargar y almacenar datos.

Características

  • Una única licencia para muchos equipos
  • Integración de equipos multifabricante via modbus (opcional)
  • Visualización de diagrama unifilar (opcional)
  • La licencia es por servidor y no depende de las variables a registrar
